Welcome to the Task Automation Tool

About

The Task Automation Tool is designed to help users schedule and automate repetitive tasks. This tool allows you to create tasks with specific descriptions and schedule them to run at specified times. The tasks are stored in an SQLite database and are executed automatically using the `node-schedule` library.

How to Use the Tool

To use the Task Automation Tool, follow these steps:

  1. Navigate to the main tool page by clicking the button below.
  2. Fill out the form with the task name, description, scheduled time, and recurrence interval.
  3. Click the "Add Task" button to schedule the task.
  4. View the scheduled tasks in the "Scheduled Tasks" section.
  5. View the completed tasks in the "Completed Tasks" dropdown.
  6. Clear all scheduled tasks by clicking the "Clear Scheduled Tasks" button.
